# Chinese AI Text Detector (Local-JS Edition)

一个完全运行在浏览器端的 **中文 AI 文本检测器**。无需后端服务器，无需 API 调用，100% 本地推理，隐私安全无忧。

---

## ✨ 特性

* **纯本地运行**：所有的计算都在用户的浏览器中完成，文本不会上传到任何服务器。
* **零依赖**：不需要大型深度学习框架，只需引入一个 `.js` 文件。
* **轻量高效**：基于优化后的卷积神经网络（CNN）架构，兼顾性能与体积。
* **隐私至上**：特别适合对数据敏感的应用场景。

---

## 🚀 快速开始

### 1. 引入脚本

将生成的 `infer.js` 放入你的项目目录，并在 HTML 中引用：

```html
<script src="path/to/infer.js"></script>

```

### 2. 调用预测函数

脚本加载后会暴露一个全局函数 `predict(text)`，它返回一个 `0` 到 `1` 之间的浮点数。

```javascript
const text = "这是一段需要检测的中文文本...";
const humanProbability = predict(text);

console.log(`人类书写概率: ${(humanProbability * 100).toFixed(2)}%`);

if (humanProbability < 0.5) {
    console.log("检测结果：该文本更有可能由 AI 生成。");
} else {
    console.log("检测结果：该文本更有可能由人类创作。");
}

```

---

## 📊 模型表现

* **适用语言**：简体/繁体中文。
* **推荐长度**：输入 **200 字以上** 的文本可以获得更准确的判断。对于极短的句子，结果仅供参考。
* **检测原理**：模型通过分析文本中的词法分布、句式特征及潜在的“机器味”进行概率评估。
